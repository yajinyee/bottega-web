# 👜 Bottega Veneta 商品目錄

> 台灣 / 日本 / 澳門 — 每日自動更新的 Bottega Veneta 商品庫存追蹤

🔗 **瀏覽網站：https://yajinyee.github.io/bottega-web/**

---

## 這是什麼？

這是一個自動化的 Bottega Veneta 精品商品目錄，每天從官網爬取最新商品資訊，包含：

- 📦 **商品資料** — 名稱、顏色、價格、上市日期
- 📏 **尺寸庫存** — 線上有貨 / 缺貨的尺寸
- 🏪 **門市庫存** — 各門市的即時庫存狀態
- 🏷️ **下架追蹤** — 已下架商品記錄（Outlet 預測）
- 💱 **多地區價格** — 台灣 NT$ / 日本 ¥ / 澳門 MOP$

## 為什麼做這個？

Bottega Veneta 官網沒有提供跨地區比價、庫存查詢、新品通知等功能。身為精品愛好者，我想要：

1. **知道什麼時候有新品上架** — 不用每天手動刷官網
2. **比較不同地區的價格** — 台灣、日本、澳門哪裡買最划算
3. **追蹤特定商品的庫存** — 想要的尺寸有貨時第一時間知道
4. **預測哪些商品可能進 Outlet** — 根據下架時間和歷史模式

所以我用 AI Agent 團隊建了一套自動化系統，這個靜態網站是其中的公開展示層。

## 資料更新頻率

| 資料類型 | 更新頻率 | 說明 |
|---------|---------|------|
| 商品列表 | 每日 10:00 | 4 地區 × 19 分類 = 76 頁 |
| 庫存快照 | 每日 06:00 | 呼叫 BV SFCC API |
| 匯率 | 每週一 08:00 | TWD/JPY/MOP/HKD |
| 靜態網站 | 爬蟲完成後 | 自動生成 + push |

## 頁面說明

| 頁面 | 說明 |
|------|------|
| `index.html` | 首頁（預設台灣） |
| `index_{region}.html` | 各地區全部商品 |
| `index_{region}_{category}.html` | 分類篩選 |
| `product/{id}.html` | 商品詳情（多角度圖 + 庫存表） |
| `delisted_{region}.html` | 已下架商品 |
| `bespoke.html` | 訂製服務資訊 |

## 技術細節

- **爬蟲**：Python + urllib（無第三方 HTTP 庫）
- **資料庫**：SQLite（WAL mode）
- **靜態生成**：Python 直接呼叫 render 函式輸出 HTML
- **部署**：GitHub Pages（每次 force push，不保留歷史）
- **圖片**：直接引用 BV CDN（不下載到 repo）
- **排程**：Python `schedule` 庫 + 自建 scheduler

## 隸屬專案

本靜態網站是 [ark-team-agent](https://github.com/yajinyee/team-agent) 多 AI Agent 團隊管理系統的一部分。Bottega-agent 是團隊中的一個 worker agent，負責精品庫存追蹤。
